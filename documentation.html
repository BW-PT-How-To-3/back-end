    Request URL Description
    POST api/users/register Registers a new user
    POST api/users/login Authenticates user
    PUT api/users/update/:id Updates a users (username, email, or role only) (required role: superadmin)
    DELETE api/users/delete/:id Currently Not working (required role: superadmin)
    GET api/users/getusers Returns All users account. (required role: superadmin)
    POST api/howto/ Adds a new howto post
    PUT api/howto/update/:id Updates a post by by id
    DELETE api/howto/delete/:id Deletes Post
    GET api/howto/getall gets all publically posted posts
    GET api/howto/getHowto/:id Returns All users account. (required role: superadmin)
